# rabin2用法

概述：

* `-I`：binary info
  ```bash
  rabin2 -I binaryFile
  ```
* `-i`：imports
  ```bash
  rabin2 -i binaryFile
  ```
* `-E`：exports
  ```bash
  rabin2 -E binaryFile
  ```
* `-l`：linked libraries
  ```bash
  rabin2 -l binaryFile
  ```
* `-z`：strings (from data section)
  ```bash
  rabin2 -z binaryFile
  ```
* `-s`：symbols
  ```bash
  rabin2 -s binaryFile
  ```
* `-S`：sections
  ```bash
  rabin2 -S binaryFile
  ```
* `-SS`：segments
  ```bash
  rabin2 -SS binaryFile
  ```
